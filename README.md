# dotfiles

> dotfiles <3 for ZSH

## Install & Setup

```sh
$ git clone https://github.com/earaujoassis/dotfiles.git ~/dotfiles
$ ~/dotfiles/bootstrap.zsh
$ ~/dotfiles/macos-setup.sh
$ ~/dotfiles/git-setup.sh
```
